SONG Table

This table will store some songs in my music library.
songs.csv populates this table.

Columns
  • SongID = Key
  • Artist = the songs artist
  • Album = the songs album
  • Title = the song title
  • Year = the year the song was released


SONGRATING Table

This table will allow me to rate the songs in my library, and record all the ratings

Columns
  • SongRatingID = Key
  • TheSongID = the song being rated
  • Rating = the rating of the song
  • Name = the name of the person rating
